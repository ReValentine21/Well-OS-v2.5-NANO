Wel-OS v2.5
📝 Описание проекта
Wel-OS — это легковесная микро-операционная система с графическим интерфейсом (GUI), разработанная для микроконтроллеров семейства Arduino и миниатюрных OLED-дисплеев. Проект создан как интерактивная экосистема, которая превращает микроконтроллер в полноценный девайс с рабочим столом, таскбаром, системными часами и встроенным софтом. Главная фишка системы — управление в реальном времени с помощью мыши и клавиатуры твоего ПК через высокоскоростной Python-сервер (Serial Bridge).
🆕 Что нового в версии 2.5
Переход на Fullscreen UI: Приложения больше не сминаются и не накладываются друг на друга. Каждая программа открывается на весь экран, имеет собственную шапку (Title Bar), кнопку сворачивания _ и быстрого закрытия X.
Физические границы экрана: Реализована система "невидимых стен" для курсора. Мышь больше не вылетает за границы матрицы 128x64 и четко упирается в края дисплея.
Исправление багов Таскбара: Проблема с пустыми квадратами решена. В нижней панели теперь отображаются уникальные текстовые иконки запущенных и свернутых утилит (>, +, *, O).
Интерактивное меню «Пуск»: Нажатие на логотип [w] открывает контекстное меню с возможностью отправить ОС в энергосберегающий режим (Sleep) или сделать честную аппаратную перезагрузку (Reboot) платы Arduino.
✨ Особенности и фишки
Анимированный Boot Screen: При старте системы отображается стильный логотип компании WEL CORP со сглаженной полосой загрузки компонентов.
Тактильный клик (Flash-отклик): Каждое нажатие на интерфейсные элементы (кнопки калькулятора, ячейки Сапёра) сопровождается мгновенной инверсией цвета для визуального подтверждения клика.
Умный Сапёр (Saper): Честное игровое поле 6x4 с подсчетом мин вокруг. При наезде на мину игра не просто выдает экран смерти RIP, но и автоматически раскрывает на карте расположение всех остальных скрытых бомб для анализа ошибок.
Глубокая кастомизация в Settings:
Secs: Включение/выключение отображения секунд на часах таскбара.
Dark: Быстрое переключение в инвертированный ночной режим (Dark Mode).
Fast: Режим ускорения курсора мыши для динамичного перемещения.
Br (Brightness): Аппаратное управление контрастностью OLED-чипа (3 уровня: Min / Mid / Max).
Максимальная оптимизация: Шина I2C разогнана до 400 кГц, что гарантирует стабильные 60 FPS, отсутствие задержек ввода и просадок кадров.
🔌 Железо и компоненты
Плата: Arduino (Uno, Nano, Pro Micro или любая другая совместимая плата на базе ATmega328p/ATmega32u4).
Дисплей: Монохромный OLED-дисплей на базе чипа SSD1306, разрешение 128x64 пикселей, интерфейс подключения I2C.
Органы управления: Физическая мышь и клавиатура твоего компьютера. Координаты и нажатия клавиш перехватываются Python-скриптом на ПК и транслируются в плату через кабель USB-Serial.
🗺️ Схема подключения (Пины для Arduino Uno/Nano)
OLED VCC -> 5V (или 3.3V в зависимости от модуля дисплея)
OLED GND -> GND
OLED SCL -> Пин A5 (или выделенный SCL пин)
OLED SDA -> Пин A4 (или выделенный SDA пин)
📚 Используемые библиотеки
Wire.h — встроенная библиотека для работы с аппаратным протоколом I2C на повышенных частотах.
Adafruit_GFX.h — библиотека для отрисовки базовых графических примитивов (линии, треугольники, шрифты).
Adafruit_SSD1306.h — низкоуровневый драйвер для управления матрицей и отправки конфигурационных команд в дисплей.
🛠️ Инструкция по установке и запуску
Сборка железа: Соедини плату Arduino и OLED-дисплей по схеме I2C, приведенной выше. Подключи плату к компьютеру по USB.
Прошивка микроконтроллера: Установи библиотеки Adafruit GFX и Adafruit SSD1306 через Менеджер библиотек в Arduino IDE. Открой файл скетча wel_os_251.ino и загрузи его в плату.
Запуск сервера управления: Установи интерпретатор Python на ПК (если еще не установлен) и необходимые зависимости (pip install pyserial pyautogui). Запусти транслирующий Python-скрипт, указав COM-порт твоей Arduino.
Готово! Система встретит тебя экраном загрузки, после чего управление полностью перейдет к мышке твоего ПК.
