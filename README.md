# 🚀 Wel-OS v2.5

Крупное обновление микро-операционной системы для Arduino и OLED-дисплея SSD1306 (128x64). В версии 2.5 мы полностью переработали логику интерфейса, исправили баги с отображением и перешли на полноэкранную архитектуру для максимального удобства.

---

## 🆕 Что нового в v2.5?

* Полноэкранные приложения (Fullscreen UI): Забудьте про слепленные и мелкие окна. Теперь приложения открываются на весь экран, снабжены удобной шапкой, кнопкой сворачивания _ и закрытия X.
* Жесткие границы для курсора: Мышь больше не «улетает» за край экрана. Реализованы невидимые стены — курсор четко упирается в рамки дисплея.
* Исправление Таскбара: Баг с пустыми квадратами исправлен. Теперь в нижней панели корректно отображаются иконки запущенных и свернутых программ (>, +, *, O).
* **Новое меню «Пуск» [w]:** Клик по кнопке в левом нижнем углу вызывает меню с возможностью отправить системSleepep** (энергосбережение) или сделRebootot** (честный аппаратный сброс Arduino)Перезапуск встроенного софта:а:**
    * *Сапёр (Saper):* Полностью переписан. Теперь это честная игра на сетке 6x4 с логичным подсчетом мин вокруг и кнопкой быстрого сброса RST.
    * *Калькулятор:* Получил удобную сетку кнопок на весь экран и поддержку ввода цифр/операторов прямо с физической клавиатуры ПК.
