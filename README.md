# 🚀 Wel-OS v2.5.1

Крупное обновление микро-операционной системы для Arduino и OLED-дисплея SSD1306 (128x64). В версии 2.5 мы полностью переработали логику интерфейса, исправили баги с отображением и перешли на полноэкранную архитектуру для максимального удобства.
🔌 Железо и компоненты
Плата: Arduino (Uno, Nano, Pro Micro или любая другая совместимая плата на базе ATmega328p/ATmega32u4).
Дисплей: Монохромный OLED-дисплей на базе чипа SSD1306, разрешение 128x64 пикселей, интерфейс подключения I2C.
Органы управления: Физическая мышь и клавиатура твоего компьютера. Координаты и нажатия клавиш перехватываются Python-скриптом на ПК и транслируются в плату через кабель USB-Serial.
🗺️ Схема подключения (Пины для Arduino Uno/Nano)
OLED VCC -> 5V (или 3.3V в зависимости от модуля дисплея)
OLED GND -> GND
OLED SCL -> Пин A5 (или выделенный SCL пин)
OLED SDA -> Пин A4 (или выделенный SDA пин)
📚 Используемые библиотеки
Wire.h — встроенная библиотека для работы с аппаратным протоколом I2C на повышенных частотах.
Adafruit_GFX.h — библиотека для отрисовки базовых графических примитивов (линии, треугольники, шрифты).
Adafruit_SSD1306.h — низкоуровневый драйвер для управления матрицей и отправки конфигурационных команд в дисплей.
🛠️ Инструкция по установке и запуску
Сборка железа: Соедини плату Arduino и OLED-дисплей по схеме I2C, приведенной выше. Подключи плату к компьютеру по USB.

## 🆕 Что нового в v2.5.1?
✨ Особенности и фишки
Анимированный Boot Screen: При старте системы отображается стильный логотип компании WEL CORP со сглаженной полосой загрузки компонентов.
Тактильный клик (Flash-отклик): Каждое нажатие на интерфейсные элементы (кнопки калькулятора, ячейки Сапёра) сопровождается мгновенной инверсией цвета для визуального подтверждения клика.
Умный Сапёр (Saper): Честное игровое поле 6x4 с подсчетом мин вокруг. При наезде на мину игра не просто выдает экран смерти RIP, но и автоматически раскрывает на карте расположение всех остальных скрытых бомб для анализа ошибок.
Глубокая кастомизация в Settings:
Secs: Включение/выключение отображения секунд на часах таскбара.
Dark: Быстрое переключение в инвертированный ночной режим (Dark Mode).
Fast: Режим ускорения курсора мыши для динамичного перемещения.
Br (Brightness): Аппаратное управление контрастностью OLED-чипа (3 уровня: Min / Mid / Max).
